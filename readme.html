Zelle® Payments for Zen Cart

Version 1.0.0

Current Support Thread at Zen Cart Forums: https://www.zen-cart.com/showthread.php?227626-Zelle-Payments-Support-Thread.

What it does

The Zelle® payment module enables your store to collect payments via a customer's Zelle app. You provide, as part of the payment module's configuration, the Zelle ID that you're registered to receive payments. That's it, pretty simple.

Version History:

  • v1.0.0, 20210108 (lat9). Initial release.

Installation

The following files are provided. Simply copy them to your Zen Cart storefront, then navigate to your admin's Modules :: Payment tool to install and configure the payment method:

  1. /includes/languages/english/modules/payment/zelle.php
  2. /includes/modules/payment/zelle.php
